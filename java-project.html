AICaptcha

Description

AICaptcha is a Java-based program that generates a CAPTCHA using 8 AI-generated images and 1 real image. Users interact with a RECaptcha-style interface, selecting the real image among the AI-generated ones. This project demonstrates proficiency in Java, GUI handling, and automated image selection logic.

Features

  • AI-Generated Image Selection: Combines 8 AI images with 1 real image to simulate a RECaptcha challenge.
  • Interactive GUI: Users click the image they believe is real; program provides visual feedback.
  • Expandable Design: Easily add new images to the pool for testing or demonstration.
  • Animated Feedback: Shows a checkmark or X when a selection is made, then closes window gracefully.

Project Structure

AICaptcha/
├─ cap2.java        # Main program file
├─ images/          # Folder containing real and AI images
└─ README.md

How to Run

Place the AICaptcha folder in your Java projects directory and run: 

javac cap2.java
java cap2
